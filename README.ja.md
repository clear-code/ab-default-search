ab-default-search
=================

アドレス帳の既定の検索クエリを定義できるようにします。

LDAPアドレス帳を使用している場合、そのアドレス帳のアドレスの一覧を見るためには、何らかの検索を行う必要があります。このアドオンを導入すると、アドレス帳が自動的に「@」で検索されるようになるため、LDAPアドレス帳でいちいち手動で検索を実施する必要が無くなります。

既定の検索クエリは変更できます。「設定エディタ（about:config）」を開き、 `extensions.ab-default-search@clear-code.com.query` の値を好みに合わせて変更してください。
